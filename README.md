## Run Locally

Clone the project

```bash
  git clone https://github.com/yashraj-n/python-gender-recognition
```

Go to the project directory

```bash
  cd python-gender-recognition
```

Install dependencies

```bash
  pip install numpy cv2 cvlib
```

Run

```bash
  python gender-detector.py
```
